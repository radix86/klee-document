什么是符号执行
=========================
符号执行就是不用具体的值去执行程序，而是选用“符号化”的变量来执行程序，这些“符号化”的变量并不代表一个具体的值，而是代表了一类具有相同特征的值。

                    var a 
                      |
                      |
                      V
                    foo(a)
                      |
                      |
                      V
                  if(a == 0)
                   __ |
                  | 
                  V
